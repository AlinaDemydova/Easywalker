Автообновление в браузере:
1) Переходим на сайт browsersync.io
2) Копируем текст из окошка
	npm install -g browser-sync


Веб-райтер, копирайтер, автор, тексты для сайта, контент-менеджер

Редактируем прямо в браузере
В консоли:
document.body.contentEditable=true;

ctrl+L = очистить консоль
или clear();

1) Находим папку с файлами нашего сайта.
2) Нажимаем на ней: Shift+правая кнопку мыши
3) Выбираем из меню "Открыть окно команд"
4) Открывается node.js 
5) Вводим в командную строку:
browser-sync start --server --files "php/*.php,css/*.css,js/*.js,html/*.html,*.*"
